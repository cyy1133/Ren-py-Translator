Codex CLI 번역 모드

Codex CLI의 `Sign in with ChatGPT` 세션을 사용해 번역합니다. API 키를 직접 넣지 않고, 파일 단위의 큰 문맥 청크로 번역해 배치 분할을 줄입니다. Windows에서는 `codex_npx.cmd` 래퍼를 기본으로 사용하고, Codex CLI가 없으면 npm으로 자동 설치를 시도한 뒤 장치 로그인 창을 엽니다.

CLI 점검 전입니다.

Gemini는 현재 문서 기준 `gemini-2.5-flash`, `gemini-2.5-flash-lite`, `gemini-2.5-pro`, `gemini-3-flash-preview`, `gemini-3.1-flash-lite-preview`, `gemini-3.1-pro-preview`를 추천 목록에 넣어 두었습니다.

Codex OAuth 권장값은 `auto-codex-economy`입니다. 짧은 UI/고정 문자열은 더 싼 모델로, 긴 대사/묘사는 더 강한 모델로 자동 라우팅하고, 문서가 하나 끝날 때마다 바로 `tl/...` 출력과 publish bundle에 반영합니다.

Publish

Ren'Py 언어 팩 / 폰트 설정

번역이 끝나면 staging 출력과 별도로 Ren'Py가 직접 읽을 수 있는 `game/tl/<lang>` 번들을 같이 생성합니다. translation layer 모드에서는 헤더 언어 코드까지 바꿔서 실제 언어 팩처럼 배치하고, source 파일 직접 분석 모드에서는 tl 팩 자동 생성이 어려워 경고만 남깁니다. 폰트를 바꾸면 현재 게임 GUI 폰트/크기와 비교해서 과하게 커지거나 작아지지 않도록 자동 보정할 수 있고, 특정 style만 따로 다른 폰트를 지정하는 경로도 열어 둡니다.

게임 분석 후 기본 GUI 폰트, 크기, 언어 스위치 훅을 여기서 요약합니다.

Style 별도 폰트 설정

특정 style 이름에만 다른 폰트를 지정할 수 있습니다. 크기를 비우면 원래 style 크기를 유지합니다.
